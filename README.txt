HCE 홈페이지 사용 안내

1. GitHub 저장소 최상위에 index.html, style.css, script.js를 업로드합니다.
2. Settings → Pages → Deploy from a branch → main → /(root) → Save를 선택합니다.
3. script.js 첫 줄의 hcecenter@example.com을 실제 문의용 이메일로 변경합니다.
4. index.html 하단의 대표자명, 사업자등록번호, 주소, 전화번호, 이메일을 실제 정보로 변경합니다.
5. 이 버전은 별도 사진 없이 디자인되어 images 폴더가 필요하지 않습니다.
6. 문의 양식은 서버 저장 방식이 아니라 사용자의 이메일 작성창을 여는 방식입니다.
